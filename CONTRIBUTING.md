# Contributing

Contributions are welcome.

This project is still in early development, so the API, structure, and behavior
may change often. If you want to contribute, keep that in mind and expect some
iteration.

You can contribute however you want:

- Open an issue
- Send a pull request
- Suggest API changes
- Report bugs
- Improve docs

Small changes and experiments are fine. If something is unclear, open the
discussion anyway.

The main thing to understand is that breaking changes are still normal at this
stage.
